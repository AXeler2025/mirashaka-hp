# mirashaka-hp

## 概要

mirashaka-hpは静的ウェブサイトです。

## 公開情報

このサイトは **GitHub Pages** によって公開されています。

- **公開URL**: https://axeler2025.github.io/mirashaka-hp/
- **リポジトリ**: https://github.com/AXeler2025/mirashaka-hp

## プロジェクト構成

```
mirashaka-hp/
├── index.html      # メインページ
├── css/
│   └── style.css   # スタイルシート
├── js/
│   └── main.js     # JavaScript
├── images/
│   └── hero.png    # 画像素材
└── docs/
    └── README.md   # 公開情報ドキュメント
```

## ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/AXeler2025/mirashaka-hp.git
cd mirashaka-hp

# ローカルサーバーで確認（例: Python）
python -m http.server 8000

# ブラウザで http://localhost:8000 にアクセス
```

## デプロイ

`main` ブランチにプッシュすると、GitHub Pagesが自動的にサイトを更新します。
