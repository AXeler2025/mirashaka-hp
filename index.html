SPACE スペース紹介

Communication

コミュニケーションスペース

面積10.8m x 16.3m
収容約100人
設備4人掛け席・ソファ席

対話を中心としたオープンスペース。什器を移動して簡易ホールとしても利用可能。交流会やワークショップ、音楽系の公演など多目的に使えます。

Share Office

シェアオフィス

面積10.8m x 16.3m
設備デスク・複合プリンター
付帯専用会議室 x 4（各10m²）

契約団体が自由に利用できるオフィススペース。専用の会議室も4室併設しています。

Creation

クリエーションスペース

面積12.6m x 10.9m
収容約50人
設備3Dプリンター・大判印刷機

製作機械を使って、自分のデザインを形にできるスペース。個人制作や制作系ワークショップに最適です。

Grand Staircase

大階段

座席約50席
設備プロジェクター・大型スクリーン・音響設備

階段状の座席から大型スクリーンの映像を観覧可能。セミナーやプレゼンテーション、映画鑑賞会、パブリックビューイングなどに。

Event Hall

イベントホール

面積21.5m x 13.9m
収容約300人
設備プロジェクター・大型スクリーン・音響設備・収納型座席

本棟最大のホール。講演会、授業、ダンス・合唱・演奏の発表や練習など幅広い用途に対応。座席の収納も可能です。

Meeting

会議室（1F・3室）

面積30m² x 1室 / 20m² x 2室
収容各8〜10人
設備長机・椅子
Communication

コミュニケーションスペース

面積16.2m x 9.0m
収容約20人
設備小上がり・スツール・掲示板・配架台

2〜4人程度の交流や対話に。掲示板や配架台で企画・団体の情報発信も可能です。

Group Work

グループワークスペース

面積24.3m x 11.7m
収容約100人
設備可動式机・椅子・ホワイトボード・電子モニター

什器を自由に組み合わせてワークスペースを構築。グループワーク型授業、ミーティング、ワークショップ、発表会に。

Presentation

プレゼンテーションスペース（4か所）

面積5.4m x 5.4m（2区画）/ 7.2m x 5.4m（2区画）
収容10〜40人
設備可動式机・椅子・ホワイトボード・電子モニター

セミナー、ゼミ、発表会、パネルディスカッションなどに。電子モニターに向けた配置で効果的な発表が可能です。

Solo Work

ソロワークスペース

座席約50席
設備机・椅子・電源・発話ブース

個人作業に集中できるスペース。発話ブースは別途利用予約が必要です。

Meeting

会議室（2F・2室）

面積各30m²
収容各8〜10人
設備長机・椅子

利用方法

自由利用

学内外を問わず、自由に入館・利用できます。

予約利用

施設の貸切や企画実施には、事前の予約申請が必要です。

ご利用いただける方

  • 学内の学生・教職員（正課・正課外）
  • 学外の非営利団体・個人
  • 学外の営利団体・個人
  • 企画運営委員会が認めた団体・個人

※特定の宗教・政治的活動、許可のない営利活動は禁止です

利用時のお願い

  • 貴重品は常に携帯してください
  • 敷地内は全面禁煙です
  • フリースペースの占有はご遠慮ください
  • 飲食可（匂いの強いものはNG・清掃をお願いします）
  • 音出し（音響・楽器・発表等）は企画申請が必要です
  • ゴミはお持ち帰りください

予約方法

HPからの予約

  1. HPで予約状況を確認
  2. 予約フォームから利用者情報・企画情報を送信
  3. 予約受理後、確認メールが届く
  4. 利用時に受付で確認メールを提示・支払い
学内者の予約はコチラ

受付からの予約

  1. 受付で予約状況を照会
  2. 利用申請書を記入して提出
  3. 企画利用の場合は企画書も提出
  4. 予約受理後、確認メールが届く
  5. 利用時に受付で確認メールを提示・支払い

留意事項

  • 予約は受付順です
  • 大学の公的行事が優先される場合があります
  • 予約は1年以内の日程に限ります
  • 同一団体・個人の同時予約は3つまでです

キャンセルポリシー

  • 前日まで無料
  • 予約時間前まで10%
  • 予約時間後100%

※変更・キャンセルはお問い合わせください
