### Phase 3: 利用案内セクション（GUIDE）の新設

**現状**: なし
**変更後**: ナビに「GUIDE（ご利用案内）」を追加し、以下の情報を掲載

#### 3-1. 利用者区分
Wordの利用者テーブル（Table 0）を反映：
- 区分1: 学内 学生・教職員（正課）
- 区分2: 学内 学生・教職員（正課外）
- 区分3: 学外 非営利団体・個人
- 区分4: 学外 営利団体・個人
- 区分5: 企画運営委員会が認めた団体・個人

※特定の宗教・政治的活動、許可のない営利活動は利用不可

#### 3-2. 利用方法
- **一般利用**: 学内外問わず自由に入場可能
- **予約利用**: 企画実施・個別利用は別途申請が必要

#### 3-3. 開館時間
- 授業開講日: 8:00-22:00
- 土日祝日（休講日）: 10:00-22:00
- 長期休暇中: 10:00-19:00
- ※事前申請で変更可能

#### 3-4. 利用のルール
- 全般的なお願い（マナー、貴重品管理、禁煙、占有禁止、営利活動禁止、宗教政治活動禁止）
- 飲食について（基本OK、匂いの強いもの注意、清掃、試食会は事前申請、ゴミ持ち帰り）
- 音出しについて（通話配慮、音響使用は企画申請必要、会話音量配慮）

### Phase 4: 予約フローセクション（RESERVATION）の新設

**現状**: フッターに「学内者の予約はコチラ」ボタンのみ
**変更後**: ナビに「RESERVATION（施設予約）」を追加

#### 4-1. 予約可能スペース一覧
Wordの Table 1 をベースにしたテーブル：
- 階層・場所・予約区分・備考（Web予約期限）
- ※料金列は今回含めない（「料金表に記載」→「お問い合わせください」等に差し替え）

#### 4-2. 予約方法（2パターン）

**HPからの予約（ステップ表示）**:
1. HPから利用したい区画の予約状況を確認
2. 予約フォームから利用者情報・企画情報を記入して送信
3. 予約受理後、確認メールが届く
4. 利用時に受付で確認メールを提示し、利用料を支払い

**受付からの予約（ステップ表示）**:
1. 受付で予約状況を照会
2. 利用申請書を記入して提出
3. 企画利用の場合は企画書も提出
4. 予約受理後、確認メールが届く
5. 利用時に受付で確認メールを提示し、利用料を支払い

#### 4-3. 申請に必要な情報

**利用者情報**:
- 名前、団体名、所属・学籍番号（学生）、住所、連絡先
- 利用場所、利用日時、利用目的

**企画情報**（企画実施利用の場合）:
- 企画名、企画内容
- 営利活動の有無
- 入場料・参加料の設定の有無
- 関係団体の有無（協賛金・営利活動の有無）
- 飲食提供の有無
- 企画書の添付

#### 4-4. 留意事項
- 予約は受付順
- 大学公的行事が優先される場合あり
- 予約は1年以内
- 同一団体・個人の同時予約は3つまで

#### 4-5. キャンセルポリシー
- 前日まで: 無料
- 予約時間前まで: 10%
- 予約時間を超えてのキャンセル: 100%
- 変更・キャンセルの連絡先: ※整備中

### Phase 5: 企画申請セクション（EVENT の拡充）

**現状**: イベント開催の簡単な案内 + Google Formリンク
**変更後**: EVENTセクションを拡充し、企画申請の詳細を追加

- 企画申請の流れ（企画書作成 → フォーム記入・添付 → 受理 → 確認メール → 利用開始）
- 営利活動の定義
- 入場料・参加料の設定について
- 金銭授受の責任について
- 飲食を伴う企画（3パターン: 参加者のみ / 無償提供 / 営利提供）
