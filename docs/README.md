- **サービス**: GitHub Pages

### GitHub Pagesについて

GitHub Pagesは、GitHubリポジトリから直接静的ウェブサイトをホスティングできる無料のサービスです。

- リポジトリへのpushで自動的にサイトが更新されます
- カスタムドメインの設定も可能です
- HTTPSが標準でサポートされています

### デプロイ方法

`main` ブランチにプッシュすると、GitHub Pagesが自動的にサイトをビルド・デプロイします。変更は数分以内に反映されます。

### ローカル開発

```bash
# リポジトリをクローン
git clone https://github.com/AXeler2025/mirashaka-hp.git

# ローカルサーバーで確認（例: Python）
python -m http.server 8000

# ブラウザで http://localhost:8000 にアクセス
```
